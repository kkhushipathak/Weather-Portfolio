# 🌤️ Weather App

A modern and responsive **Weather App** built using **HTML, CSS, and JavaScript**. The application integrates **WeatherAPI** to provide real-time weather information, current location weather, and a 5-day weather forecast through an interactive and user-friendly interface.

---

## 🚀 Features

- 🔍 Search weather by city name
- 📍 Get weather using current location
- 🌡️ Real-time temperature updates
- ☁️ Current weather conditions
- 💧 Humidity information
- 🌬️ Wind speed details
- 🌡️ Feels-like temperature
- 📅 5-Day Weather Forecast
- 🌙 Dark/Light Mode
- 🕒 Recent Search History
- 🎨 Dynamic background based on weather conditions
- 📱 Fully Responsive Design

---

## ▶️ How to Run the Project

1. Clone this repository.
2. Open the project folder.
3. Replace the API key in `script5.js` with your own WeatherAPI key.
4. Open `weather-portfolio.html` in your browser or run it using Live Server.
